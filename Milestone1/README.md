# FitPlan AI — Personalized Fitness Plan Generator  
## Milestone 1: Front-End Development

### Objective
To build a web application that collects user fitness details and calculates BMI before integrating the AI model.

### BMI Formula
BMI = weight (kg) / (height in meters)²

### Steps Performed
- Created fitness profile form using Streamlit
- Implemented input validation
- Converted height from cm to meters
- Implemented BMI calculation logic
- Classified BMI into categories
- Displayed user's name with BMI result
- Deployed application on Hugging Face Spaces

### Technologies Used
- Python
- Streamlit
- Hugging Face Spaces
