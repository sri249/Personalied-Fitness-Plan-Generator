# FitPlan AI — Personalized Fitness Plan Generator
## Milestone 2: Core AI Model Integration

### Objective
Integrate a Large Language Model (LLM) to generate personalized workout plans using user fitness inputs.

### Model Used
Qwen/Qwen2.5-7B-Instruct via Hugging Face Inference API

### Prompt Design
Dynamic prompt constructed using:
- Name
- Age
- Gender
- BMI Category
- Fitness Goal
- Fitness Level
- Available Equipment

### Steps Performed
- Integrated Hugging Face LLM using API
- Built modular prompt generator
- Implemented BMI calculation
- Generated personalized workout plans
- Deployed on Hugging Face Spaces
